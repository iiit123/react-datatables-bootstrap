React Datatables with Bootstrap

React-datatable is a react bootstrap component which can be very easily integrated with your code.

It can be easily configured and has many features like search, sorting, pagination. It is compatible with audio files as well as images. Extra actions buttons can be included for modifiying the table

The following piece of code creates the entire datatable for your webpage.

< DataTable

tags= {tags in the json which have to be displayed}

actions={[{'name': 'button name to be displayed in datatable', 'function': function

defined in parent class, 'btn_class':'bootstrap class of the button'}]}

data = {json object which has to be displayed} />

React datatable which has been created by taking a json object as input. Very useful in displaying JSON objects which are loaded from the server. Check out the source code
